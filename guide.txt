### 4. üóÑÔ∏è SQLite Database

**Purpose:** Query structured data

**When to Use:**
- Employee information
- Project data
- Structured queries
- Data analysis

**Database Schema:**

**Employees Table:**
```sql
CREATE TABLE employees (
    id INTEGER PRIMARY KEY,
    name TEXT,
    department TEXT,
    salary INTEGER,
    hire_date TEXT
)
```

**Projects Table:**
```sql

    id INTEGER PRIMARY KEY,
    project_name TEXT,
    budget INTEGER,
    status TEXT,
    start_date TEXT
)
```

**Example Queries:**
```
‚úÖ "Show all employees in Engineering"
‚úÖ "What's the average salary by department?"
‚úÖ "List all active projects"
‚úÖ "Which employees earn more than 90000?"
‚úÖ "Show projects with budget over 400000"
```

**Sample SQL Queries:**
```sql
-- All employees
SELECT * FROM employees

-- Average salary by department
SELECT department, AVG(salary) as avg_salary 
FROM employees 
GROUP BY department

-- Active projects
SELECT * FROM projects WHERE status = 'Active'

-- High earners
SELECT name, salary FROM employees 
WHERE salary > 90000 
ORDER BY salary DESC

-- Project budget summary
SELECT status, SUM(budget) as total_budget 
FROM projects 
GROUP BY status
```

**Safety:**
- Only SELECT queries allowed
- Read-only access
- No data modification

## üõ†Ô∏è Troubleshooting

### Common Issues

**1. Tool Not Available**
```
Error: DuckDuckGo tool unavailable

Solution:
pip install duckduckgo-search
```

**2. Database Locked**
```
Error: database is locked

Solution:
Close other connections to agent_database.db
```

**3. File Not Found**
```
Error: File not found: report.txt

Solution:
Use absolute path: /full/path/to/report.txt
Or place in current directory
```

**4. No Results from ArXiv**
```
No papers found

Solution:
- Check spelling
- Try broader keywords
- Specify category (e.g., "cs.AI")
```

**5. Agent Times Out**
```
Agent stopped due to time limit

Solution:
- Simplify query
- Use specific tool explicitly
- Increase max_execution_time in code
```
